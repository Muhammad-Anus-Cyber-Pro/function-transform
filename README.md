# 🏠 House Price Prediction using Linear Regression

## 📌 Overview
This project predicts house prices using **Linear Regression** with proper preprocessing, feature engineering, and transformations.  
The main focus is to explore how **log transformation on skewed features** affects model performance.

---

## ⚙️ Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn
